Advanced animation effects library for Oxygen Builder with scroll triggers, background effects, and customizable attributes.

== Description ==

Oxy Animation is a powerful WordPress plugin that brings professional CSS animations to Oxygen Builder. Create stunning visual effects with an extensive library of animations and background effects.

**Key Features:**

* **80+ Animation Effects** - Complete Animate.css library including bounce, fade, slide, zoom, and more
* **Background Effects** - Ken Burns, Background Banner, and Color Background animations
* **Customizable Attributes** - Control duration, delay, repeat, and trigger options
* **Smart Triggers** - On Scroll, On Hover, On Click, or On Page Load
* **Visual Effects Library** - Browse and preview all effects with live examples
* **Copy & Paste Ready** - Generated code for Oxygen Builder, HTML, CSS, and JavaScript
* **Fully Responsive** - Works perfectly on all devices and screen sizes
* **Lightweight** - Pure CSS animations with minimal JavaScript

**Background Effects:**

* **Ken Burns Effect** - Cinematic zoom and pan for background images with 10 direction variations
* **Background Banner** - Sliding background animations with 8 directional movements
* **Color Background** - Smooth color transitions with customizable timing and direction

**Animation Categories:**

* Attention Seekers (bounce, flash, pulse, shake, swing, etc.)
* Bouncing Entrances (bounceIn, bounceInDown, bounceInUp, etc.)
* Fading Entrances (fadeIn, fadeInDown, fadeInLeft, fadeInRight, etc.)
* Flippers (flip, flipInX, flipInY, etc.)
* Rotating Entrances (rotateIn, rotateInDownLeft, etc.)
* Sliding Entrances (slideInLeft, slideInRight, slideInUp, etc.)
* Zooming Entrances (zoomIn, zoomInDown, zoomInUp, etc.)
* Background Effects (Ken Burns, Background Banner, Color Background)

**Perfect For:**

* Landing pages with scroll-triggered animations
* Hero sections with background effects
* Call-to-action elements
* Product showcases
* Portfolio websites
* Marketing pages

**Developer Friendly:**

* Clean, documented code
* WordPress coding standards
* Extendable architecture
* Custom animation support via filters

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/oxy-animation/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Oxygen Builder
4. Add animation classes and attributes to your elements
5. Browse the Effects Library at Oxy Animation > Effects Library

== Frequently Asked Questions ==

= Does this work with Oxygen Builder? =

Yes! This plugin is specifically designed for Oxygen Builder and integrates seamlessly with it.

= Can I use these animations without Oxygen Builder? =

Yes, the animations are pure CSS and can be used with any theme or page builder by adding the CSS classes.

= Are the animations responsive? =

Yes, all animations are fully responsive and work on all devices.

= How do I trigger animations on scroll? =

Set the trigger attribute to "scroll" or use `data-oxy-anim-trigger="scroll"` on your element.

= Can I customize animation speed and duration? =

Yes, you can customize duration, delay, repeat count, timing function, direction, and fill mode through attributes.

= Do the background effects work with gradients? =

Yes, the Color Background effect works with solid colors, gradients, and any CSS background.
